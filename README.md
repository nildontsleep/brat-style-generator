### Prerequisites

Ensure you have:

* Python 3.10 or later
* Required modules: `pywebview`, `screeninfo` (install via `pip install pywebview screeninfo`)

### Clone the Repository

```bash
git clone https://github.com/nildontsleep/brat-style-generator.git
```
